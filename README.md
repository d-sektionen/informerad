# Usage

<!-- usage -->

```sh-session
$ npm install -g informerad
$ informerad COMMAND
running command...
$ informerad (-v|--version|version)
informerad/1.0.0 linux-x64 node-v8.12.0
$ informerad --help [COMMAND]
USAGE
  $ informerad COMMAND
...
```
